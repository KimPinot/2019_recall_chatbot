# Recall_ChatBot

* 일시 : 2019-11-30 (13:00) ~ 2019-12-01 (18:00)
* 장소 : 중앙대학교
* 개발 분야 : 챗봇 구현(기획, 디자인, 프로그래밍)
* 결과 : 우수상
* 스택 : 자바스크립트 (ES6 형식으로 개발함) + 미들웨어 (Koa.JS + promise request.js)
